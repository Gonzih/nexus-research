# Meta-Layer Initiative — Document Index

**Organization:** ISOC Nevada / The Meta-Layer Initiative
**Archive Date:** 2026-05-07
**Source:** https://govhub.live/doc/all/
**Total Documents:** 27

All documents are governance drafts (ML-Drafts) for the Meta-Layer Initiative, a project to design an open, federated governance layer for the web. Most are "Desirable Properties" (DP) — normative specifications for how the meta-layer should behave.
